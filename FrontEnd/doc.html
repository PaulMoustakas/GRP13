SpotiMap

API Dokumentation

Här återfinns all information som är av vikt för att kunna söka på ett specifikt lands topp 50 lista på spotify utifrån enbart namnet på det specifika landet. Med andra ord, se nedan för vidare beskrivning och dokumentation för att använda SpotiMaps API.




Kort API beskrivning

Detta API gör det möjligt att få ut en spotify spellista genom dess unika ID, detta nås med hjälp av ett specifikt land skrivet på svenska. Användaren byter ut {country} i GET funktionen till de land vars topp 50 lista på Spotify de eftersöker. Vårt API accepterar indata i form av JSON och returnerar JSON objekt tillbaka till användaren.


GET / {country}

Returnerar ett ID för en specifik spellista från spotify, utifrån landet.

Se exempel för returdata till höger

  • Indata: -

  • Returdata (exempel):
  • {
  • "countryName":":Sverige",
  • "top50Playlist":"37i9dQZEVXbLoATJ81JYXz"
  • }
GET / {artist}

Returnerar information om en specifik artist

Se exempel för returdata till höger

  • Indata: -

  • Returdata (exempel):
  • {
  • "artistName":":Drake",
  • "artistInfo":"Aubrey Drake Graham[5] (born October 24, 1986) is a Canadian rapper, singer, songwriter, actor, and entrepreneur...."
  • }
